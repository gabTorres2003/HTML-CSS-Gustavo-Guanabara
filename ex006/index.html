Estilos e Significados

Semântica

A semântica é o que define a categoria de cada item. Temos diferentes maneiras de representar a mesma coisa em HTML, mas a maneira semântica é a mais indicada para um código limpo.

Forma

A forma é a maneira gráfica de representar estilos em HTML, o que acaba tendo um significado raso, pois os estilos são definidos pelas CSS, por esse motivo muitas tags de estilo estão sendo extintas e entrando em desuso. Justamente pela substituição da forma pelo significado.
